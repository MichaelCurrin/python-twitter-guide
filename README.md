## Remote setup

Fork as your own Github project.

To go settings and make sure Github Pages is enabled.

Check the _environment_ tab to check on the deploy.


## Run locally

### Installation

Install NPM.

Install DocsifyJS globally.

```sh
$ npm i -g docsify
```

Clone the repo.


### Usage

```sh
$ docsify serve docs
```
